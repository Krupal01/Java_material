§1 · Primitives & Wrapper Classes Q1–Q30
1What are the 8 primitive data types in Java? List them with sizes.
THEORY

byte(1B), short(2B), int(4B), long(8B), float(4B), double(8B), char(2B), boolean(JVM-dependent, conceptually 1 bit).

2Default value of int, boolean, double, char, and Object reference?
THEORY

int=0, boolean=false, double=0.0, char='\u0000' (null char), Object reference=null.

3What is autoboxing and unboxing?
THEORY

Autoboxing: automatic conversion of primitive to wrapper (int→Integer). Unboxing: wrapper to primitive (Integer→int). Done by the compiler automatically.

4What will Integer.valueOf(127) == Integer.valueOf(127) return?
THEORY🔥 TRICKY
Integer a = 127; Integer b = 127; System.out.println(a == b); // true

true — Integer caches values from -128 to 127. Both a and b refer to the same cached object.

5What will Integer.valueOf(128) == Integer.valueOf(128) return?
THEORY🔥 TRICKY
Integer a = 128; Integer b = 128; System.out.println(a == b); // false

false — 128 is outside the Integer cache range. Two different heap objects are created. Use .equals() for value comparison.

6Output of: int x = 5; System.out.println(x++ + ++x)?
THEORY🔥 TRICKY
int x = 5; System.out.println(x++ + ++x); // 12

12 — x++ uses 5 then x becomes 6; then ++x makes x=7. So 5+7=12.

7What happens assigning long to int without casting?
THEORY

Compile-time error: incompatible types: possible lossy conversion from long to int. Explicit cast required.

8Result of 1/2 in Java?
THEORY🔥 TRICKY
System.out.println(1/2); // 0

0 — both operands are int, so integer division is performed. Use 1.0/2 or (double)1/2 to get 0.5.

9Range of byte? What happens if you exceed it?
THEORY

byte range: -128 to 127. Exceeding it causes overflow — wraps around. E.g., (byte)128 = -128.

10Difference between float and double literals?
THEORY

double is the default floating-point type. 3.14 is a double. For float, suffix with 'f': 3.14f. Assigning double to float without cast causes compile error.

11What does System.out.println(0.1 + 0.2) print?
THEORY🔥 TRICKY
System.out.println(0.1 + 0.2); // 0.30000000000000004

Floating-point numbers cannot represent 0.1 and 0.2 exactly in binary (IEEE 754). Use BigDecimal for precision.

12Difference between == for primitives vs objects?
THEORY

For primitives, == compares values. For objects, == compares memory references. Use .equals() for object content comparison.

13What happens: Integer x = null; int y = x;?
THEORY🔥 TRICKY

NullPointerException at runtime — unboxing null causes NPE.

14Result of: System.out.println('a' + 1)?
THEORY🔥 TRICKY
System.out.println('a' + 1); // 98

98 — char 'a' has ASCII value 97. 97+1=98 (int result). To print 'b', use: System.out.println((char)('a'+1)).

15What is widening type conversion?
THEORY

Automatic promotion from smaller to larger type: byte→short→int→long→float→double. E.g., int i=5; double d=i; works without cast.

19Output: System.out.println(10 == 10.0)?
THEORY🔥 TRICKY
System.out.println(10 == 10.0); // true

true — int 10 is widened to double 10.0 before comparison.

20Difference between & and && operators?
THEORY

& is bitwise AND (evaluates both sides always). && is logical AND (short-circuit: if left side is false, right side is NOT evaluated).

21Result of ~5 in Java?
THEORY🔥 TRICKY
System.out.println(~5); // -6

-6. Bitwise NOT flips all bits. Formula: ~n = -(n+1). So ~5 = -6.

25Output: System.out.println(true + false)?
THEORY🔥 TRICKY

Compile-time error — boolean values cannot be used with the + operator in Java.

27Result of Integer.MAX_VALUE + 1?
THEORY🔥 TRICKY
System.out.println(Integer.MAX_VALUE + 1); // -2147483648

-2147483648 (Integer.MIN_VALUE). Integer overflow wraps around silently in Java.

28What is a wrapper class? Why is it needed?
THEORY

Wrapper classes (Integer, Double, Boolean, etc.) wrap primitives as objects. Needed for: Collections, generics, null representation, utility methods like parseInt.

30What will (byte)(127+1) print?
THEORY🔥 TRICKY
System.out.println((byte)(127+1)); // -128

-128. 127 is the max byte value. Adding 1 causes overflow and wraps to -128.

THEORY

A special memory area in the heap (Java 8+). When a string literal is created, JVM checks the pool. If found, returns same reference. Saves memory by deduplication.

  • String pool feasibility

Unified architecture for representing/manipulating collections. Main interfaces: Collection, List, Set, Queue, Deque. Map is NOT a Collection.

132Collection vs Collections?

Collection (interface): root interface for List, Set, Queue. Collections (utility class): static methods like sort(), shuffle(), synchronizedList().

134Initial capacity and load factor of HashMap?
THEORY

Initial capacity: 16 buckets. Load factor: 0.75 (rehash when 75% full). On overflow, capacity doubles.

135What happens when two keys have same hashCode in HashMap?
THEORY

Hash collision — same bucket. Java 7: chained as LinkedList. Java 8+: converts to Red-Black Tree when bucket size > 8.

136list.remove(1) on List<Integer> [1,2]?
THEORY🔥 TRICKY
List<Integer> list = new ArrayList<>();

System.out.println(list); // [1]

[1] — list.remove(1) calls remove(int index), removing index 1 (value 2). Use list.remove(Integer.valueOf(1)) to remove by value.

137How to iterate a List and remove elements safely?
THEORY

Use Iterator's remove() method. Or removeIf() (Java 8+). Direct removal in for-each causes ConcurrentModificationException.

Iterator<String> it = list.iterator();
while (it.hasNext()) {
    if (it.next().isEmpty()) it.remove(); // safe
}
141Fail-fast vs fail-safe iterators?
THEORY

Fail-fast: throws ConcurrentModificationException if modified during iteration (ArrayList, HashMap iterators).
Fail-safe: operate on a snapshot copy (CopyOnWriteArrayList, ConcurrentHashMap iterators).

144Comparable vs Comparator?
THEORY

Comparable (java.lang): natural ordering, class implements compareTo() — single sorting sequence. Comparator (java.util): external, separate class or lambda — multiple sorting sequences, more flexible.

145List.of("a","b").add("c") output?
THEORY🔥 TRICKY

UnsupportedOperationException — List.of() (Java 9+) creates an IMMUTABLE list.

151What happens when a mutable object used as HashMap key is mutated?
THEORY🔥 TRICKY

Entry becomes unreachable — mutation changes hashCode, key lands in different bucket. Entry is effectively lost. Always use immutable objects as Map keys.

164containsKey() vs contains() in Map?
THEORY🔥 TRICKY

Map has NO contains() method. Map has containsKey(key) and containsValue(value). Collection has contains(). Common trick question.

§8 · String Methods — Coding Q301–Q320
301Check if a given string is a palindrome.
EASYString
Task: Write a method to check if a given string is a palindrome (reads the same forwards and backwards).
boolean isPalindrome(String s) {
    int l = 0, r = s.length() - 1;
    while (l < r) {
        if (s.charAt(l++) != s.charAt(r--)) return false;
    }
    return true;
}

307Find the first non-repeating character in a string.
MEDIUMString
Task: Find first non-repeating character and return its index. Return -1 if none.
int firstNonRepeating(String s) {
    Map<Character, Integer> freq = new LinkedHashMap<>();
    for (char c : s.toCharArray())
        freq.merge(c, 1, Integer::sum);
    for (int i = 0; i < s.length(); i++)
        if (freq.get(s.charAt(i)) == 1) return i;
    return -1;
}
312Longest substring without repeating characters.
MEDIUMString
int lengthOfLongestSubstring(String s) {
    Map<Character,Integer> map = new HashMap<>();
    int max = 0, left = 0;
    for (int right = 0; right < s.length(); right++) {
        char c = s.charAt(right);
        if (map.containsKey(c)) left = Math.max(left, map.get(c)+1);
        map.put(c, right);
        max = Math.max(max, right - left + 1);
    }
    return max;
} // Sliding window O(n)
314String compression: "aaabbc" → "a3b2c1".
MEDIUMString
String compress(String s) {
    StringBuilder sb = new StringBuilder();
    int i = 0;
    while (i < s.length()) {
        char c = s.charAt(i); int cnt = 0;
        while (i < s.length() && s.charAt(i) == c) { i++; cnt++; }
        sb.append(c).append(cnt);
    }
    return sb.length() < s.length() ? sb.toString() : s;
}
319Find all permutations of a given string.
HARDString
List<String> permute(String s) {
    List<String> result = new ArrayList<>();
    permHelper("", s, result);
    return result;
}
void permHelper(String prefix, String rem, List<String> res) {
    if (rem.isEmpty()) { res.add(prefix); return; }
    for (int i = 0; i < rem.length(); i++)
        permHelper(prefix + rem.charAt(i),
                   rem.substring(0,i) + rem.substring(i+1), res);
}
320Longest palindromic substring — expand-around-center.
HARDString
String longestPalindrome(String s) {
    int start = 0, maxLen = 0;
    for (int i = 0; i < s.length(); i++) {
        for (int len : expand(s, i, i), expand(s, i, i+1)) {
            if (len > maxLen) { maxLen = len; start = i - (len-1)/2; }
        }
    }
    return s.substring(start, start + maxLen);
}
int expand(String s, int l, int r) {
    while (l >= 0 && r < s.length() && s.charAt(l)==s.charAt(r)) { l--; r++; }
    return r - l - 1;
}

337Two Sum — find two indices that add to target.
MEDIUMArrays
int[] twoSum(int[] nums, int target) {
    Map<Integer,Integer> map = new HashMap<>();
    for (int i = 0; i < nums.length; i++) {
        int complement = target - nums[i];
        if (map.containsKey(complement))
            return new int[]{map.get(complement), i};

void mergeSort(int[] arr, int l, int r) {
    if (l >= r) return;
    int m = (l + r) / 2;
    mergeSort(arr, l, m);
    mergeSort(arr, m+1, r);
    merge(arr, l, m, r);
}
void merge(int[] arr, int l, int m, int r) {
    int[] tmp = Arrays.copyOfRange(arr, l, r+1);
    int i=0, j=m-l+1, k=l;
    while (i<=m-l && j<=r-l) arr[k++] = tmp[i]<=tmp[j] ? tmp[i++] : tmp[j++];
    while (i<=m-l) arr[k++] = tmp[i++];
    while (j<=r-l) arr[k++] = tmp[j++];
}

392Sliding window maximum using a Deque.

int[] maxSlidingWindow(int[] nums, int k) {
    int[] res = new int[nums.length - k + 1];
    Deque<Integer> dq = new ArrayDeque<>(); // stores indices
    for (int i = 0; i < nums.length; i++) {
        if (!dq.isEmpty() && dq.peekFirst() < i-k+1) dq.pollFirst();
        while (!dq.isEmpty() && nums[dq.peekLast()] <= nums[i]) dq.pollLast();
        dq.offerLast(i);
        if (i >= k-1) res[i-k+1] = nums[dq.peekFirst()];
    }
    return res;
} // O(n)

String decodeString(String s) {
    Deque<Integer> counts = new ArrayDeque<>();
    Deque<StringBuilder> strs = new ArrayDeque<>();
    StringBuilder cur = new StringBuilder();
    int k = 0;
    for (char c : s.toCharArray()) {
        if (Character.isDigit(c)) k = k*10 + (c-'0');
        else if (c == '[') { counts.push(k); strs.push(cur); cur=new StringBuilder(); k=0; }
        else if (c == ']') {
            int rep = counts.pop();
            StringBuilder prev = strs.pop();
            for (int i=0; i<rep; i++) prev.append(cur);
            cur = prev;
        } else cur.append(c);
    }
    return cur.toString();
}

class Trie {
    Map<Character, Trie> children = new HashMap<>();
    boolean isEnd = false;

    void insert(String word) {
        Trie node = this;
        for (char c : word.toCharArray()) {
            node.children.putIfAbsent(c, new Trie());
            node = node.children.get(c);
        }
        node.isEnd = true;
    }
    boolean search(String word) {
        Trie node = this;
        for (char c : word.toCharArray()) {
            if (!node.children.containsKey(c)) return false;
            node = node.children.get(c);
        }
        return node.isEnd;
    }
    boolean startsWith(String prefix) {

        for (char c : prefix.toCharArray()) {
            if (!node.children.containsKey(c)) return false;
            node = node.children.get(c);
        }
        return true;
    }
}
477Detect cycle in course prerequisites graph using DFS.

boolean canFinish(int n, int[][] prereqs) {
    Map<Integer,List<Integer>> graph = new HashMap<>();
    for (int[] p : prereqs) graph.computeIfAbsent(p[0], k->new ArrayList<>()).add(p[1]);
    // 0=unvisited, 1=in-progress, 2=done
    int[] state = new int[n];
    for (int i=0; i<n; i++) if (hasCycle(i, graph, state)) return false;
    return true;
}
boolean hasCycle(int node, Map<Integer,List<Integer>> g, int[] state) {
    if (state[node] == 1) return true;
    if (state[node] == 2) return false;
    state[node] = 1;
    for (int nei : g.getOrDefault(node, List.of()))
        if (hasCycle(nei, g, state)) return true;
    state[node] = 2;
    return false;
}

        while (!pq.isEmpty() && feed.size() < 10) feed.add(pq.poll()[1]);
        return feed;
    }
