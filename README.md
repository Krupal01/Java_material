## 🚀 GitHub Pages Setup

1. Create a repo (e.g. `Java_material`)
2. Copy all `.html` files to the repo root
3. Go to **Settings → Pages → Source → Deploy from branch → main / root**
4. Your site will be live at `https://yourusername.github.io/Java_material/`
